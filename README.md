Структура директорий
controllers - Контроллеры FXML
services - Логика приложений (сервисный слой)
models - Модели данных
utils - Вспомогательные классы и утилиты
fxml - FXML-файлы интерфейса
styles - CSS-стили
images - Изображения, иконки
